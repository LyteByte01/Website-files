What is Python?

Python is a high level programming language and is the most used programming language out there with nearly 30% of people saying it was their main programming language. Python is used in many places around you such as self checkouts and specific systems around you. One example you can see for yourself is Google.com, they use python for their searches. You will rarely see python used as a front-end language (what is displayed on screen) and more as a back-end lenguage (the processing behind) some other backend languages are C#, PHP and Kotlin. Now, lets get started!

Page Setup

The first thing we want to do if get the Python file ready to edit. If you are using replit it was described on how to set up a python project. However if you are using the installed python on your computer then you will want to follow the instructions below:


1. Search for and open IDLE



2. Click file, new for a new document, DO NOT WRITE CODE IN THE FIRST WINDOW THAT APPEARS! This is your python shell and is used to display your code when you run it, writing code in here will not work properly and cannot be saved.

Once you have completed you should have two windows: one with python shell and another one that is blank (picture below shows. We will be using the blank window to be programming in.



Now, we are going to learn our first bit of syntax (a piece of syntax is a bit of code) and it is a comment. Comments don't affect the way your program functions, and they can not be seen by the end user. They are helpful for seperating code and explaining what it does. Try and always use these in your code as it is a good habit to get into and is helpful for you and others when working in a team.
