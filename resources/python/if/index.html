If statements!

If statements are very and I mean very useful if you ever want to decide what the correct answer is in your program. They can create, differnt outputs for the length of your string or what number you entered. They allow you to create validation loops which are used in the real world today. Like when you try and enter a certain password after creating an account IF the password is less than 5 characters it won't let you use it. Now you understand the use of if statements let me introduction ELSE statements, they are a lot simpler than if statements because it means if the user enters anything ELSE than what is in your if statement it is wrong.
