Once you've tried that out we are going to look at user inputs, basically were anything can go inside the brackets when running the program. 


      user_input = input("Enter your Name: ")

What is user_input you're wondering? Well it is a variable (a variable is something that stores values). In this case user_input will be whatever you enter but will always be a string (a string is a sequence of characters). This is because if you don't add "int(" it can't recognise numbers but just remember if you add two brackets to one side you have to balance the otherside.
