What are Variables?

Variables they can store any character, number, float (decimal number) or user inputs. An example of this is shown below. Feel free to copy this into replit or python and try yourself!



The first variable you see "x" is storing an integer or a number. You know when there is a variable when you have the character or characters then an equals sign and whatever you want to store so in this case "1". This is called declaring a variable.


The second variable you see "y" is storing a string meaning the variable is storing words and characters. Thi can be used for storing users inputs.


The third variable is storing a float number this means the number has a decimal place like "1.1111". This can be used when doing maths in your program and getting storing precise results.


The fourth and final variable is storing a boolean variable this means it can either be "True" or "False". This can be used in checking loops but you'll learn that later on!
